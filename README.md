# 1% Better Static Website

Đây là bộ file web tĩnh cho trang 1% Better.

## Cấu trúc file

- `index.html`: nội dung chính của trang
- `styles.css`: màu sắc, bố cục, responsive mobile
- `script.js`: menu mobile
- `README.md`: hướng dẫn sử dụng

## Cách chỉnh nội dung

1. Mở thư mục bằng Cursor hoặc VS Code.
2. Sửa chữ trong `index.html`.
3. Sửa màu, khoảng cách, font trong `styles.css`.
4. Mở `index.html` bằng trình duyệt để xem trước.

## Cách đưa lên GitHub

1. Tạo repository mới trên GitHub, ví dụ: `1better-website`.
2. Upload 3 file: `index.html`, `styles.css`, `script.js`.
3. Commit thay đổi.

## Cách deploy bằng Vercel

1. Vào Vercel.
2. Chọn Add New Project.
3. Import repository GitHub vừa tạo.
4. Framework Preset để là Other hoặc Static.
5. Deploy.

## Gợi ý thay phần form

Form trong bản này chỉ là mẫu giao diện. Khi dùng thật, có thể đổi sang:

- Google Form
- MailerLite
- ConvertKit
- Formspree
- Tally

## Lưu ý nội dung

Vì 1% Better có màu tâm lý ứng dụng, nên nội dung nên giữ theo hướng giáo dục và tự quan sát. Không viết theo hướng chẩn đoán, chữa trị, hoặc hứa kết quả tuyệt đối.
